# **App Name**: ForgeFit

## Core Features:

- Custom Workout Planner: Users can create and customize weekly workout plans with exercises from various modalities (weights, calisthenics, running, cycling) including goals like 'Reps', 'Sets', or 'Time'.
- Pre-Loaded Exercise Database: Access a database of over 50 common exercises with clear titles, detailed instructions, and placeholder images/GIFs. Users can browse and add these to their weekly plan.
- Weekly Check-off & Reset: Mark exercises or entire daily workouts as complete. An automated system clears all completed checks every Monday at 00:00, resetting the weekly plan while retaining configured exercises.
- Performance Tracker: Record and view progress metrics like maximum load for exercises, fastest 'Km time', total distance for runs, and body weight/measurements over time.
- Progress Visualization: Visualize fitness evolution over months with simple and intuitive charts based on performance data.
- Nutrition & Hydration Tracking: Track daily protein intake by marking goal achievement and monitor water consumption with a customizable daily goal and interactive counter.
- AI Workout Instruction Assistant: Leverage a generative AI tool to provide expanded, alternative, or refined execution instructions for custom exercises or suggest variations based on user-defined parameters.

## Style Guidelines:

- Primary color: Energetic, clear blue (#267AD9), inspired by 'cobalt blue' to evoke focus and motivation.
- Background color: A very light, subtly tinted blue-gray (#EEF3F8), providing a clean and expansive canvas for content.
- Accent color: A deep, invigorating cyan (#0C85A3) for interactive elements and key information, ensuring contrast and drawing attention.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, techy, and motivating feel.
- Body text: 'Inter' (sans-serif) for its clean, neutral, and highly readable qualities across various content types, aligning with Material Design principles.
- Use modern, clean, and vector-based icons that adhere to Material Design guidelines for clarity and intuitive navigation.
- Adopt a clean and intuitive layout, prioritizing clear hierarchy, sufficient white space, and logical flow for an optimal user experience, consistent with Material Design.
- Implement subtle and purposeful animations for transitions between screens, check-off confirmations, and chart loading, enhancing user feedback and app's modern feel.
